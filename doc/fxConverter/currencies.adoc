= Currencies

Uses https://ratesapi.io/[Foreign exchange rates] API

[NOTE]
====
Those routes will only be available if _fxConverter_ service is enabled
====

== List currencies

*GET* _/fxConverter/currencies_

=== Result

Result will be an array of symbols
